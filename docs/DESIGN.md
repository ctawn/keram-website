# Design

## Direction

“The Journey, Curated” uses midnight navy, warm ivory, copper, editorial typography, generous space, and restrained route/horizon motifs. Arabic is the primary design language.

## Reference patterns

The Saudi-market reference `myrtn.com` informed three conventions: Arabic-first navigation, destination/month discovery in the hero, and persistent access to human help. Keram applies these patterns through an original visual system and original copy. Regional category visibility, licensing details, and verified WhatsApp/phone actions should be added when official business data is supplied.

Almosafer informed bilingual parity and visible support; Booking.com informed concise trust proof; Expedia informed transparent package composition and price qualifiers.

## Accessibility

- Semantic landmarks, headings, labels, live form status, and keyboard-accessible controls
- Minimum 44px primary controls and visible native focus behavior
- RTL/LTR applied per locale container
- Motion removed when `prefers-reduced-motion` is enabled
- Text alternatives describe meaningful photographic content

Photography currently uses licensed Unsplash references and must be replaced or formally approved before final production content sign-off.
